Скрипт предназначен для проведения турниров "Своя игра". Несколько раз использовался на интенсивной школе "Перспектива". Поддерживает обычные вопросы и вопросы "Кот в мешке".

С помощью страницы create_game.htm генерируем код игры. Для этого надо знать все вопросы, количество и названия команд. Полученный код можно сохранить в файл, чтобы его использовать в следующих турнирах.

Для проведения игры запускаем страницу index.htm, в поле ввода копируем весь код игры и нажимаем "Начать игру".
Стандартные найстроки такие, что наибольшая красота достигается, если развернуть браузер в полный экран (кнопка F11).
